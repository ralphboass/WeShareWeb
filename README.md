# WeShare - Ride Sharing Web App

A static HTML website for WeShare, a ride-sharing service that helps people in Los Angeles share rides and skip traffic.

## Project Structure

- `static-html/` - Contains the main website files
  - `index.html` - Main HTML file
  - `styles.css` - Custom styles
  - `script.js` - JavaScript functionality
  - `assets/` - Images and other static assets

## Getting Started

To view the website locally:

1. Navigate to the `static-html` directory
2. Use a local server to serve the files. You can use Python's built-in server:
   ```bash
   cd static-html
   python3 -m http.server 8000
   ```
3. Open `http://localhost:8000` in your browser

## Features

- Responsive design that works on all devices
- Clean, modern UI with smooth animations
- Easy-to-use ride sharing interface
- Built with vanilla HTML, CSS, and JavaScript
